# HZH 方案工具

这是一个内部使用的 Streamlit 工具，用于管理 POI 点位，并生成可编辑的英文客户方案 PowerPoint。

## 功能

- 录入英文方案标题、客户人数、天数晚数、路线城市、预算参考和英文简介
- 根据客户类型、偏好标签、行程强度和 POI 数据库自动生成路线初稿
- 在生成后的右侧预览中编辑每日英文行程安排和每日点位
- 管理 POI 点位数据库，包括新增、查看、编辑、删除、筛选和搜索
- 支持从 Excel / CSV 导入 POI 数据
- 每天可自动推荐多个 POI，并可继续手动增加、删除、替换；系统会带出英文名称、英文介绍、建议时长和成本区间
- 上传封面图、每日行程图和视觉亮点图
- 未上传图片时，PPT 中保留可编辑的图片占位符
- 生成可编辑的英文 `.pptx` 客户方案

## 语言规则

- `interfaceLanguage = "zh-CN"`：工具操作界面统一使用中文。
- `outputLanguage = "en"`：导出的客户版 PPT 统一使用英文。
- POI 数据底层字段名保留英文 key，例如 `name_cn`、`name_en`、`description_en`、`price_min`、`price_max`。
- 前端界面展示中文字段名，例如「中文名称」「英文名称」「英文介绍」「最低成本价」「最高成本价」。
- 生成 PPT 时优先调用并只展示英文客户字段，例如 `name_en`、`description_en`。

## 本地运行

```bash
pip install -r requirements.txt
streamlit run app.py
```

打开终端显示的本地地址，通常是：

```text
http://localhost:8501
```


## 批量导入现有 POI 表格

项目支持从 Excel 或 CSV 批量导入 POI 数据。推荐文件路径：

```text
data/HZH医旅POI多维标签数据库-价格完善版.xlsx
```

也可以使用：

```text
data/poi_database.csv
```

### 在页面中导入

1. 打开「POI 点位管理」。
2. 在「导入 POI 数据」上传 Excel 或 CSV。
3. 查看「字段映射结果」和「导入数据预览」。
4. 选择导入方式：
   - 「只新增不覆盖」：保留已有同编号点位，只添加新点位。
   - 「覆盖同编号点位」：同 `poi_id` 的点位使用新表内容更新。
   - 「替换全部现有数据」：用新表完全替换当前点位库。
5. 点击「确认导入」。

### 使用脚本导入

也可以在终端运行：

```bash
python poi_importer.py data/HZH医旅POI多维标签数据库-价格完善版.xlsx --output data/poi_database.csv --mode replace
```

`--mode` 可选：

- `replace`：替换输出文件。
- `append`：只新增不覆盖。
- `overwrite`：覆盖同编号点位。

导入脚本会自动识别字段、生成缺失的 `poi_id`、去重、解析价格区间、解析时长，并把标签统一为分号分隔格式。

## POI 点位管理

POI 数据库位于：

```text
data/poi_database.csv
```

CSV 模板位于：

```text
outputs/poi_template.csv
```

### 导入 POI

1. 打开「POI 点位管理」。
2. 下载 `poi_template.csv`。
3. 按模板填写点位数据，每行一个点位。
4. `tags` 和 `suitable_for` 等多值字段建议用英文分号或中文分号分隔。
5. 上传 CSV。
6. 选择追加到现有数据库，或替换现有数据库。

### 在方案中使用 POI

1. 打开「方案生成」。
2. 在左侧填写方案基础信息、客户类型、行程偏好、行程强度和预算参考。
3. 点击「确定并生成方案初稿」。
4. 系统会根据路线城市、客户类型、偏好标签、点位类型、建议时长和成本信息自动推荐每日 POI。
5. 在右侧「每日点位选择」继续手动增加、删除或替换点位。
6. 检查「价格核算 / 报价预览」。
7. 点击「生成英文方案 PPT」。

生成的 PPT 会使用右侧当前行程和当前点位选择结果；客户版文件不显示后台空状态提示。

## 图片规则

- 封面图、每日行程图、视觉亮点图均可上传。
- 未上传图片时，PPT 会保留浅灰色、虚线边框、可编辑的图片区域，不显示后台占位符文案。
- 上传图片会按比例裁剪填充，不会拉伸变形。
- 如果 POI 的 `image_placeholder` 填写的是本地图片路径，系统会尝试作为每日图片的备用图。

## 备注

- 客户版 PPT 内容应保持英文输入。
- 中文字段主要用于内部管理和工作人员识别。
- 左侧预算只作为内部参考；客户版报价来自右侧「价格核算 / 报价预览」。POI 成本小计按 POI 数据中的币种计算，默认 CNY。
